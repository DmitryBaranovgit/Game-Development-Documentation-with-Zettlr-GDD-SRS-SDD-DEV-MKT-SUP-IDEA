# GDD-Визуальный стиль

## Графика
- Плоский минималистичный стиль.
- Платформы: разные текстуры и цвета для разнообразия.

## Анимации
- Мягкие переходы для персонажа.
- Враги с плавными движениями.

## Цветовая палитра
Основные цвета: голубой, белый, золотой.
